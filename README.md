# NuBank-Dolar-Quotation

## Description
----------------
Simple project to calculate USD Dollar to BRL Real with fees and taxes.

## Dependencies
----------------
- jekyll

## Development
----------------
Instructions to build and run

#### Install dependency
`gem install bundler jekyll --user-install`

#### Run
`bundle exec jekyll serve`
